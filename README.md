pttrnl
======

A front-end starter kit creating pattern libraries.
https://pttrnl.patrickwenzel.com <https://pttrnl.patrickwenzel.com>

Dependencies
------------

- Node.js / NPM
- Gulp
- Sass
- [Nunjucks](https://mozilla.github.io/nunjucks/)
- browserSync

Initialize
----------

Install dependencies...

	npm install

Run tasks...

	gulp

Build compile...

	gulp build
